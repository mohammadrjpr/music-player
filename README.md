# Music Player PWA


## امکانات

- پخش موسیقی
- نصب روی موبایل
- حالت آفلاین
- اکولایزر متحرک
- QR Code
- طراحی تاریک
- مناسب اندروید و دسکتاپ


## استفاده

1. فایل آهنگ را داخل:

assets/song.mp3

قرار دهید.


2. تصویر کاور را:

assets/cover.jpg

قرار دهید.


3. QR Code را:

assets/qrcode.png

قرار دهید.


4. پروژه را روی هاست یا لوکال سرور اجرا کنید.


## نصب

در مرورگر Chrome گزینه:

Install App

را انتخاب کنید.
